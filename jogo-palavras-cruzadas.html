Palavras Cruzadas

Jogo em desenvolvimento

As palavras cruzadas são um clássico dos jogos de raciocínio. Preencha os espaços com as palavras corretas para completar o desafio!

Como jogar:

  1. Leia a dica que aparece na horizontal ou vertical
  2. Digite a palavra que corresponde à dica
  3. Use o teclado para inserir as letras
  4. Complete todas as palavras para vencer
